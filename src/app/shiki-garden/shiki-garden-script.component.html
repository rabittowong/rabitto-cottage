自動討伐程序 v3.20
最後更新: 11 August, 2016
測試瀏覽器: Firefox 47.0
腳本功能
1. 討伐 - 根據選擇的地圖與難度循環討伐
2. 賣札牌 - 討伐地圖達一定回數時退回庭園賣卡
3. 買基本符 - 當文超過100萬時購入基本符
4. 補充AP - AP不足時自動使用行動藥(使用順序: 20AP->50AP->80AP)
5. 補充HP - HP不足時自動使用神酒瓶(HP只會在進入討伐地時檢查)
6. 刷經驗筆 - 刷指定數量的經驗筆(注意: 與一弓箭之類的掉落也會計算在內)
7. 刷活動物品 - 補充AP時同時換取試練活動裏指定行數的物品
注意事項
1. 腳本開始時需待在庭園/地圖上才能抓到座標
2. 建議持有充足的行動藥
3. 建議持有少量的神酒瓶
4. 此腳本並不相容IE跟Google Chrome
